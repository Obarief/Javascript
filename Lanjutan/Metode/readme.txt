execution context, hoisting, scope

execution context = erdapat 2 fase (1 fase pembuatan dan 1 fase eksekusi)
fase pembuatan akan membuat nilai awal undifined dalam suatu tempat 
fase eksekusi akan memasukkan nilai dari yang telah ditentukan jika tidak ada di lokal maka mencari ke window

hoisting = pengangkatan (pendeklarasian variabel berada di bawah setelah memasukkan nilai ke variabel) (hanya dapat berlaku jika menggunakan var bukan let atau const)

scope = pengecekan variabel dimulai dari scope lokal/function baru ke global/window
